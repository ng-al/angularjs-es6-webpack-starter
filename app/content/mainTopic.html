Main

/app/main creates the application Angular module and "global" assets. Everything is setup as a consequence of import "main.es6"

main.es6 performs the following tasks:

  1. Imports the top-level stylesheet. Note that main.less imports the variables from bootstrap.
  2. Imports ngRoute and tabs from the installed node_modules.
  3. Imports the routing config block, the routing provider, and the left navigation component from their respective files.
  4. Builds the application module, injecting ngRoute, bootstrap UI tabs, and the left nav feature.
  5. Registers the routing config block and the routing provider with the module.
